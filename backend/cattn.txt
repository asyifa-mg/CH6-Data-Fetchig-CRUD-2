
Langkah buat hubungkan frontend dan backend dengan react untuk tugas CRUD-react binar academy:
=============================
1. Bikin folder backend, lalu masuk ke patch atau foldernya melalui "cd. \backend

2.setelah itu inslall package.json --> npm init -y 

3. setelah itu install modul yang isinya modul express,mysql2,sequelize dan cors dengan cara buka terminal --> npm i express mysql2 sequelize cors

3. lalu tambahkan dalam package.json di atas main ---> "type":"module"
   catatan: guna ditambahkan yaitu supaya bisa digunakan ES6 syntax untuk mengimport dan export module

4.lalu buat folder index.js--> isi dengan import module express dan cors dan tambahkan server atau port berapa yang dipakai

5.lalu buka terminal--> dan ketikkan node .\index.js
